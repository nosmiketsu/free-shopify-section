## üõ†Ô∏è How to Use

### For `.liquid` Sections:
1. Copy the `.liquid` file into your `sections` directory in your theme.
2. Customize schema settings and styling as needed.
3. Use the Shopify Theme Editor to drag-and-drop it into your layout.

### For `HTML/CSS/JS` Sections:
These are structured starting templates. To convert them into Shopify sections:
- Wrap your HTML in `{% schema %}` and `{% endschema %}`
- Move styles to `theme.css` or inline
- Add dynamic settings as needed
